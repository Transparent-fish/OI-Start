    > Trie 修复：init 时正确清理根节点并置零 tot。
  - 文档
    > 新增并完善 `docs/USER.md`，包含各模块使用示例与注意事项。
  - 说明
    > Seg 目前支持 '*' 与 '+' 两种区间操作；'-' 可通过加负值模拟，'/' 未在复合懒标记中支持。请在使用前仔细阅读 docs/USER.md。
